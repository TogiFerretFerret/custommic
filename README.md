# Custom Analog Studio XLR Microphone 

A hardware-hacking engineering project to design, prototype, and manufacture a high-fidelity large-diaphragm analog XLR studio microphone from the schematic level up.

## Project Architecture
- **Transducer Core:** 34mm Gold-Sputtered Center-Terminated Condenser Capsule (RK-47 Profile)
- **Analog Preamp Subsystem:** Discrete Schoeps-style JFET impedance-matching emitter-follower circuit board (2-Layer Custom Printed PCB)
- **Power & Shielding:** 48V DC external Phantom Power delivery over a balanced 10-foot Mogami W2549 line with isolated alloy chassis Faraday shielding.
- **Diagnostic Stack:** FNIRSI 1014D 2-Channel Desktop Bench Oscilloscope calibration.

## Diagnostic Plan
Once components arrive, the oscilloscope will be deployed to trace signal integrity across the analog traces, map the 48V DC biasing rails, and evaluate transient wave behavior under high-frequency acoustic loads.
